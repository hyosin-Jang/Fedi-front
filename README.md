# Fedi-front

## 기능
- 카메라로 얼굴 사진 촬영 후 사진 업로드 미리보기
- 유사도 분석 결과보기, 신고할 사진 선택
- 네트워크 맵으로 좋아요/리트윗 누른 계정 확인
- 이메일 입력 후 트위터 신고 RPA 호출
